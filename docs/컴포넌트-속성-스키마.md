# 컴포넌트 속성 스키마

ContentBuilder 플러그인 전체의 속성을 DB로 관리하기 위한 스키마 정의.

---

## DB 테이블 구조 (제안)

```
components         — 컴포넌트 메타
component_settings — ⚙ 설정 패널 속성 (톱니바퀴)
component_fields   — ✏ 콘텐츠 편집 패널 속성 (연필)
component_items    — 반복 항목 (메뉴, 배너 슬라이드 등)
```

---

## 1. `components` — 컴포넌트 메타

| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | VARCHAR(50) PK | `data-cb-type` 값 (예: `quick-banking`) |
| label | VARCHAR(100) | 패널 표시 이름 |

| plugin_js | VARCHAR(300) | JS 파일 경로 |
| plugin_css | VARCHAR(300) | CSS 파일 경로 |
| version | VARCHAR(20) | 플러그인 버전 |
| sort_order | INT | 패널 표시 순서 |

### 데이터

## 3. `component_fields` — ✏ 콘텐츠 편집 패널 속성

우측 패널 **연필(✏)** 클릭 시 나타나는 텍스트/URL 편집 필드.
반복 항목(메뉴, 슬라이드)은 아래 `component_items`로 분리.

| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | BIGINT PK | |
| component_id | VARCHAR(50) FK | |
| key | VARCHAR(100) | 필드 식별 키 |
| label | VARCHAR(100) | 편집 패널 표시 이름 |
| field_type | ENUM | `text` `textarea` `url` `image_url` `image_file` `color` `select` |
| dom_selector | VARCHAR(200) | 연결된 DOM 선택자 |
| dom_attr | VARCHAR(50) | 속성 수정 시 해당 attribute (없으면 textContent) |
| placeholder | VARCHAR(200) | 입력 힌트 |
| is_repeatable | BOOLEAN | 반복 항목 여부 |

## 4. `component_items` — 반복 항목

메뉴, 배너 슬라이드처럼 개수가 가변적인 항목들.

| 컬럼 | 타입 | 설명 |
|---|---|---|
| id | BIGINT PK | |
| component_id | VARCHAR(50) FK | |
| item_group | VARCHAR(100) | 항목 그룹 키 (예: `navItems`, `slides`) |
| label | VARCHAR(200) | 항목 표시 텍스트 |
| url | VARCHAR(500) | 링크 URL |
| sub_label | VARCHAR(200) | 부제목 또는 설명 |
| value | VARCHAR(200) | 수치값 (금리, 환율 등) |
| icon_bg | VARCHAR(20) | 아이콘 배경 색상 |
| icon_color | VARCHAR(20) | 아이콘 색상 |
| item_type | VARCHAR(50) | 항목 분류 (branch/atm, savings/deposit/loan 등) |
| extra | JSON | 기타 속성 (badge, phone 등 컴포넌트별 추가 필드) |
| sort_order | INT | 순서 |

### 항목 그룹별 사용 필드

| component_id | item_group | 사용 컬럼 |
|---|---|---|
| top-nav | navItems | label, url |
| top-nav | drawerItems | label, url |
| top-nav | dropdownItems | label, url |
| quick-banking | items | label, url, icon_bg, icon_color |
| product-gallery | slides | label(상품명), sub_label(설명), value(금리), item_type(예금/적금/대출), url |
| exchange-board | items | label(통화명), value(매도율), sub_label(매입율), extra(변동폭, 방향) |
| branch-locator | items | label(지점명), sub_label(주소), extra(영업시간, 전화번호), item_type(branch/atm) |
| promo-banner | slides | label(제목), sub_label(설명), extra(배지텍스트, CTA텍스트, 배경색), url |
| auth-center | items | label(서비스명), sub_label(서비스설명), url, item_type(cert/otp 등) |
| product-menu | items | label, url |
| site-footer | links | label, url |

## 타입 정의 요약

```
field_type 허용값:
  text        — 단일 텍스트 (input type="text")
  textarea    — 여러 줄 텍스트
  url         — 링크 주소 (input type="url")
  image_url   — 이미지 URL (input type="url" + 미리보기)
  image_file  — 로컬 파일 업로드 (input type="file" → base64)
  color       — 색상 (input type="color")
  boolean     — 체크박스 (input type="checkbox")
  number      — 숫자 + 범위 슬라이더 (input type="number" + range)
  select      — 드롭다운 (select + options JSON)
```
