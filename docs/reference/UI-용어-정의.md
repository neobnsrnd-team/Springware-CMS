# UI 명칭 및 위치 용어 정의

개발 중 소통할 때 사용하는 공통 용어를 정리합니다.
이 명칭으로 요청하면 정확하게 이해합니다.

---

## 에디터 페이지 (`/edit`) 전체 구조

```
┌─────────────────────────────────────────────────────────────────┐
│                        [상단 네비바]                              │
│  Springware CMS  |  IBK | 하나 | KB | 신한 | 우리 | NH  |  [HTML][미리보기][저장]  │
├─────────────────────────────────────────────────────────────────┤
│  [CB 툴바]  ← ContentBuilder 자체 툴바 (에디터가 자동 생성)       │
├──────────────┬──────────────────────────────┬───────────────────┤
│              │                              │                   │
│  [블록 패널] │       [에디터 캔버스]         │   [우측 패널]     │
│              │                              │                   │
│  IBK 컴포넌트│  ← 드래그앤드랍 영역 →       │  (블록 클릭 시    │
│  기본 블록   │                              │   설정/편집 표시) │
│              │  [행]  [행]  [행] ...         │                   │
│              │                              │                   │
└──────────────┴──────────────────────────────┴───────────────────┘
```

---

## 각 영역 명칭

### 상단 네비바
- **위치**: 페이지 최상단 고정 바 (`position: fixed; top: 0`)
- **구성**: 로고 | 은행 탭 | 액션 버튼 (HTML·미리보기·저장)
- **코드 위치**: `EditClient.tsx` → `<nav>` 태그
- **요청 예시**: "상단 네비바에 탭 추가해줘", "저장 버튼 색 바꿔줘"

### 은행 탭
- **위치**: 상단 네비바 중앙
- **역할**: IBK demo / 하나 demo / KB demo ... 클릭 시 해당 은행 페이지로 이동
- **요청 예시**: "은행 탭에 SC제일 추가해줘"

### 액션 버튼
- **위치**: 상단 네비바 우측
- **구성**: `HTML` · `미리보기` · `저장`
- **요청 예시**: "액션 버튼에 '초기화' 추가해줘"

---

### CB 툴바 (ContentBuilder 툴바)
- **위치**: 상단 네비바 바로 아래, ContentBuilder가 자동 생성
- **역할**: 텍스트 서식, 색상, AI 코드챗 등 ContentBuilder 기본 기능
- **주의**: 이 영역은 ContentBuilder 라이브러리가 제어하므로 직접 수정 어려움
- **요청 예시**: "CB 툴바 아래 여백 조정해줘"

---

### 블록 패널 (스니펫 패널)
- **위치**: 에디터 좌측, `+` 버튼 클릭 시 열림
- **역할**: 드래그하거나 클릭해서 캔버스에 추가할 블록 목록
- **구성**:
  - **IBK 섹션**: 상단에 표시되는 금융 컴포넌트 8종 (커스텀 SVG 썸네일)
  - **기본 섹션**: ContentBuilder 기본 제공 블록들
- **코드 위치**: `content-plugins.js` → `var data_basic.snippets[]`
- **요청 예시**: "블록 패널 상단에 새 컴포넌트 추가해줘"

### 스니펫
- **정의**: 블록 패널에 표시되는 개별 블록 템플릿 (썸네일 이미지 + HTML)
- **요청 예시**: "환율 스니펫 HTML 수정해줘"

---

### 에디터 캔버스
- **위치**: 페이지 중앙 흰 영역
- **역할**: 블록을 배치하고 편집하는 메인 작업 공간
- **구성 단위**: 행(Row) > 열(Column) > 블록(Block) 순으로 중첩
- **요청 예시**: "캔버스에 배경색 추가해줘", "캔버스 최대 너비 바꿔줘"

### 행 (Row)
- **정의**: 캔버스 안의 가로 한 줄 단위
- **이동**: 행 좌측 `⠿` 드래그 핸들로 순서 변경
- **요청 예시**: "행 사이 간격 조정해줘"

### 금융 컴포넌트 블록
- **정의**: Springware CMS가 제공하는 금융 도메인 특화 컴포넌트. 두 종류로 구분됨
  - **순수 HTML 컴포넌트**: `data-component-id="<name>-<viewmode>"` 속성 보유. ContentBuilder 인라인 편집(더블클릭 텍스트 수정, 이미지 교체) 가능
  - **플러그인 컴포넌트**: `data-cb-type="plugin-name"` 속성 보유. JS 런타임 의존으로 플러그인 유지 (`exchange-board`, `branch-locator`, `loan-calculator`, `product-gallery`, `promo-banner`)
- **요청 예시**: "환율 플러그인 블록에 GBP 추가해줘", "퀵뱅킹 메뉴 아이콘 수정해줘"

---

### 우측 패널
- **위치**: 에디터 우측 사이드바, 블록 클릭 시 나타남
- **구성**:

| 아이콘 | 명칭 | 역할 | 대상 |
|---|---|---|---|
| ✏ (연필) | **컨텐츠 편집 패널** | 블록 내 항목 추가·삭제·텍스트 수정 | 플러그인 컴포넌트 전용 |
| ⚙ (톱니바퀴) | **설정 패널** | 색상·컬럼 수 등 플러그인 settings 값 | 플러그인 컴포넌트 전용 |

> **순수 HTML 컴포넌트**는 ContentBuilder 네이티브 인라인 편집(더블클릭)으로 직접 수정합니다. 구조화 콘텐츠(SVG 아이콘, iframe 등) 수정이 필요한 경우 `<a>` 클릭 시 `#divLinkTool`에 주입된 커스텀 버튼으로 편집 패널을 엽니다.

- **요청 예시**: "컨텐츠 편집 패널에 항목 추가 버튼 만들어줘", "설정 패널에 폰트 크기 옵션 추가해줘"

## 뷰 페이지 (`/view`)

```
┌──────────────────────────────────┐
│                                  │
│   [렌더링 영역]                   │
│   저장된 HTML + 플러그인 활성화    │
│                                  │
└──────────────────────────────────┘
```

- **렌더링 영역**: `dangerouslySetInnerHTML`로 저장된 HTML 출력
- **Runtime**: ContentBuilderRuntime이 `init()` 호출 → 플러그인 유지 컴포넌트(`exchange-board`, `branch-locator`, `loan-calculator` 등) 활성화. 순수 HTML 변환 컴포넌트는 별도 활성화 불필요
- **요청 예시**: "뷰 페이지에 헤더 추가해줘", "렌더링 영역 최대 너비 바꿔줘"

## 플러그인 코드 구조 명칭

```js
export default {
    name: 'plugin-name',          // 플러그인 ID
    displayName: '표시 이름',      // 우측 패널 제목

    settings: { },                // 설정값 → 설정 패널(⚙)에 표시

    editor: {
        openContentEditor: fn     // 컨텐츠 편집 패널(✏) UI 반환
    },

    mount: fn,                    // 뷰 페이지·에디터에서 실행
    unmount: fn                   // 정리
};
```

| 명칭 | 요청 예시 |
|---|---|
| `settings` | "settings에 배경색 옵션 추가해줘" |
| `openContentEditor` | "편집 패널에 링크 입력 필드 추가해줘" |
| `mount` | "mount에서 실시간 데이터 fetch 추가해줘" |
