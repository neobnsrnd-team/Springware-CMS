## 결정 사항

Next.js App Router 표준 **전략 A** (app 외부에 프로젝트 파일 보관)를 채택.

- `app/` 폴더는 라우팅 전용 (`page.tsx`, `route.ts`, `layout.tsx`)
- React 컴포넌트 → `src/components/<도메인>/`
- 정적 데이터·설정 → `src/data/`
- 유틸리티·헬퍼 → `src/lib/`

**이유**: 에디터·파일 관리·미리보기 등 기능 간 공유 요소가 많고, 컴포넌트 위치 규칙을 단순하게 유지하기 위함.

## 미활용 Next.js 특수 파일 (향후 필요 시 추가)

| 파일 | 역할 |
|------|------|
