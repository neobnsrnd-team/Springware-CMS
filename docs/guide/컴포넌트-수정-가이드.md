# 금융 컴포넌트 수정 가이드

컴포넌트 종류에 따라 수정해야 할 파일이 다릅니다.

---

## 컴포넌트 종류 구분

| 종류 | 대상 | 초기 HTML 출처 |
|------|------|---------------|
| **순수 HTML** | app-header, product-menu, auth-center, media-video, site-footer, product-gallery, promo-banner | `scripts/migrate-*.ts` → DB |
| **플러그인** | exchange-board, branch-locator, loan-calculator | `public/assets/plugins/<name>/index.js` |

---

## 순수 HTML 컴포넌트 수정

### 어떤 파일을 수정하는가

```
수정 목적                        파일
─────────────────────────────────────────────────────────
초기 드롭 HTML 변경             scripts/migrate-<name>-to-html.ts
                                  └─ CARDS / SLIDES / HTML 상수 수정
                                  └─ npx tsx scripts/migrate-<name>-to-html.ts 실행

구조화 콘텐츠 편집 UI 변경      src/components/edit/<Name>Editor.tsx
(SVG 아이콘, iframe, select 등)   └─ DOM 직접 수정 로직

에디터 편집 진입 버튼 변경      src/components/edit/EditClient.tsx
                                  └─ injectCustomButtonsToLinkTool()
                                  └─ updateLinkToolBtnVisibility()
```

> `public/assets/plugins/<name>/index.js` 는 이미 Runtime 등록에서 제거됨.
> 수정해도 에디터/뷰어에 반영되지 않습니다.

### 수정 후 DB 반영

```bash
npx tsx scripts/migrate-<name>-to-html.ts
```

- 기존 저장 페이지에는 영향 없음 (새로 드롭하는 컴포넌트부터 적용)
- 기존 페이지도 반영하려면 해당 블록을 삭제 후 재추가

---

## 플러그인 컴포넌트 수정

### 어떤 파일을 수정하는가

```
수정 목적                        파일
─────────────────────────────────────────────────────────
초기 HTML / 편집 패널 UI 변경   public/assets/plugins/<name>/index.js
스타일 변경                     public/assets/plugins/<name>/style.css
```

`index.js` 구조:

```js
export default {
    settings: { /* 설정 패널(⚙) 옵션 */ },
    editor: {
        openContentEditor: fn  // 편집 패널(✏) UI
    },
    mount: fn,    // 에디터·뷰어에서 마운트 시 실행
    unmount: fn,
};
```

> DB에 저장된 초기 HTML도 변경하려면 마이그레이션 스크립트를 작성해 실행합니다.
> 패턴: `scripts/migrate-<name>-to-html.ts` 참고

---

## 파일별 역할 요약

```
scripts/migrate-<name>-to-html.ts     신규 드롭 시 표시되는 초기 HTML 정의
src/components/edit/<Name>Editor.tsx  커스텀 편집 패널 UI (순수 HTML 전용)
src/components/edit/EditClient.tsx    편집 버튼 주입, 플러그인 Runtime 등록
src/components/view/ViewClient.tsx    뷰어 플러그인 Runtime 등록
public/assets/plugins/<name>/         플러그인 컴포넌트 JS + CSS
```
