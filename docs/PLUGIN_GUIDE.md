# Springware CMS — 바이브코딩 요청 가이드

> Claude에게 컴포넌트 작업을 시킬 때 어떻게 말하면 되는지를 정리한 문서입니다.

## 현재 존재하는 금융 컴포넌트 목록

| 우측 패널 표시 이름 | Claude에게 말할 때 | 주요 기능 |

## 에디터 캔버스에서 쓰는 용어

캔버스에서 컴포넌트(블록)를 클릭하면 위에 플로팅 툴바가 뜹니다.
ContentBuilder에서 **Row Tool(행 도구)** 이라고 부릅니다.

```
┌──────────────────────────────┐
│  ↕ 이동  │  ⋮ 더보기  │  🗑 삭제  │  ← Row Tool
└──────────────────────────────┘
┌──────────────────────────────────────┐
│         퀵뱅킹 메뉴 컴포넌트          │
└──────────────────────────────────────┘
```

- **이동(↕)** — 드래그로 위치 변경
- **더보기(⋮)** — 복제 / ⚙ 설정 / ✏ 편집 메뉴
- **삭제(🗑)** — 블록 제거

저한테 말할 때: "row tool", "캔버스 편집 버튼", "더보기 버튼" 등 편한 대로 말해도 됩니다.

---

## 이 프로젝트에서 "컴포넌트"가 뭔지 한 줄 요약

우측 패널에서 캔버스로 드래그하는 블록 하나 = 컴포넌트 하나.
컴포넌트는 **시각적 모양(CSS)** + **인터랙션(JS)** + **에디터에서 수정할 수 있는 UI(editor)** 로 이루어져 있습니다.

## 새 컴포넌트 만들기

### 최소한 이것만 말해주면 됩니다

```
[컴포넌트 이름] 컴포넌트 새로 만들어줘.
어떤 역할인지: (한 줄 설명)
에디터에서 수정 가능해야 하는 것: (목록)
```

**예시:**
```
"공지사항 배너" 컴포넌트 새로 만들어줘.
어떤 역할인지: 긴급 공지 문구를 강조 배경으로 띄우는 상단 띠 배너
에디터에서 수정 가능해야 하는 것: 배경색, 공지 텍스트, 링크 URL
```

### 외부 데이터를 불러와야 하는 경우

```
"공지사항 배너" 컴포넌트 새로 만들어줘.
어떤 역할인지: ...
API: GET /api/notices 로 공지 목록 불러오기
API 응답 형식: [{ id, title, url }]
API 없을 때: 기본 텍스트 보여주기
```

## 기존 컴포넌트 수정하기

### 보이는 것(스타일) 바꾸기

```
[컴포넌트명] 에서 [바꿀 것] 수정해줘.
```

**예시:**
```
quick-banking 아이콘 배경색 파란계열 → 초록계열로 바꿔줘
loan-calculator 탭 버튼 모서리 더 둥글게 해줘
top-nav 모바일 드로어 배경 흰색 → 다크 네이비로 바꿔줘
```

### 동작(인터랙션) 바꾸기

```
[컴포넌트명] 에서 [현재 동작] → [바꿀 동작] 으로 수정해줘.
```

**예시:**
```
quick-banking 아이템 클릭하면 지금은 아무것도 안 되는데,
href 링크로 실제 이동하도록 바꿔줘
```

```
product-gallery 자동 슬라이드 지금 3초인데 5초로 늘려줘
```

```
exchange-board 환율 갱신 주기 지금 60초인데 30초로 바꿔줘
```

### 에디터에서 수정 가능한 항목 추가하기

"에디터에서 수정" = 에디터 캔버스에서 컴포넌트 클릭했을 때 우측에 뜨는 패널에서 바꿀 수 있는 것

```
[컴포넌트명] 에서 [항목] 도 에디터에서 수정할 수 있게 해줘
```

**예시:**
```
product-gallery 각 카드의 배지 텍스트(예: "적금", "예금")도
에디터에서 수정할 수 있게 해줘
```

```
loan-calculator "상담 신청하기" 버튼 텍스트랑 링크 URL도
에디터에서 바꿀 수 있게 해줘
```

```
top-nav 로고 이미지를 텍스트 대신 이미지 파일로 바꿀 수 있게 해줘
```

### 설정 옵션 추가하기

"설정 옵션" = ⚙ 아이콘 누르면 나오는 패널에서 조절하는 것 (색상, 숫자, 토글 등)

```
[컴포넌트명] 설정에 [옵션명] 추가해줘.
어떤 타입인지: 색상 / 숫자 / 켜기끄기
기본값: ...
```

**예시:**
```
quick-banking 설정에 "아이콘 배경색" 색상 옵션 추가해줘. 기본값 #e8f0fe
```

```
loan-calculator 설정에 "기본 탭" 옵션 추가해줘.
대출/예금/적금 중 하나 선택, 기본값 대출
```

## 자주 쓰는 요청 패턴

### 항목 추가/삭제/순서변경이 되는 리스트 편집

quick-banking 메뉴 아이템처럼 리스트를 에디터에서 관리하고 싶을 때:

```
[컴포넌트명] 에서 [항목들] 을
에디터에서 추가/삭제/순서변경 할 수 있게 해줘.
각 항목에서 수정 가능한 것: 이름, 링크 URL (등등)
```

### 실시간 API 데이터 연동

```
[컴포넌트명] 에 실시간 데이터 연동해줘.
API 주소: GET /api/xxx
응답 형식: { ... }
데이터 표시 위치: .my-list 안에
갱신 주기: 30초마다
API 실패하면: 기존 하드코딩 데이터 그대로 보여줘
```

### 완전히 새로운 API 엔드포인트도 같이 만들어야 할 때

```
[컴포넌트명] 만들어줘.
백엔드 API도 같이 만들어야 해.
API: GET /api/[name]
반환해야 하는 데이터: [설명]
데이터 소스: 하드코딩 / 외부 API fetch / DB (해당하는 것)
```

## 참고할 기존 컴포넌트

헷갈릴 때 아래를 참고 컴포넌트로 언급하면 그 방식대로 만들어줍니다.

| 하고 싶은 것 | 참고 컴포넌트 |
|---|---|
| 리스트 항목을 추가/삭제/편집 | `quick-banking` |
| 로고, 텍스트, URL 같은 기본 정보 편집 | `top-nav` |
| 실시간 API 데이터 + 갱신 | `exchange-board` |
| 슬라이드/캐러셀 | `product-gallery` |
| 탭 전환 + 계산 로직 | `loan-calculator` |
| 카드 리스트 (클릭 가능) | `auth-center` |

**예시:**
```
"즐겨찾기 메뉴" 컴포넌트 만들어줘.
quick-banking 방식으로 항목 추가/삭제/편집 되게 해줘.
```

## 기본 블록 (우측 패널 "기본 블록" 탭)

"기본 블록"은 두 종류가 있습니다.

### 종류 1: HTML 스니펫 블록 (단순 레이아웃)

`content-plugins.js`에 정의된 순수 HTML 블록입니다.
**특징**: JavaScript 동작이 없고, 텍스트·이미지·링크를 더블클릭으로 직접 편집합니다.

이런 것들: 히어로 배너, 특징(Features) 카드, CTA(Call-to-Action), 사진+텍스트, 팀 소개 등

**수정 요청 예시:**
```
기본 블록 히어로 배너의 제목 폰트 크기 60px → 40px으로 줄여줘
기본 블록 CTA 배경색을 파란색 그라데이션으로 바꿔줘
```

---

### 종류 2: 플러그인 기반 블록 (동작 있음)

`public/assets/plugins/` 디렉터리에 JS + CSS로 구현된 플러그인 블록입니다.
**특징**: 금융 컴포넌트와 동일한 구조 (`data-cb-type`, `settings`, `editor`, `mount`, `unmount`).

| Claude에게 말할 때 | 우측 패널 표시 이름 | 주요 기능 | 에디터에서 편집 가능한 것 | 설정(⚙) 옵션 |
|---|---|---|---|---|
| `accordion` | Accordion / FAQ | 아코디언 펼치기/닫기 | 질문·답변 텍스트, 항목 추가/삭제/순서 | 여러 항목 동시 열기, 첫 항목 기본 열기 |
| `faq` | Accordion / FAQ | 아코디언 (accordion과 동일) | 질문·답변, 항목 관리 | allowMultiple, openFirst, 모서리 반경 |
| `callout-box` | Callout Box | 안내/성공/경고/오류 강조 박스 | 제목·본문 직접 클릭 편집 | 스타일(info/success/warning/error) |
| `simple-stats` | Simple Stats | 숫자 카운팅 애니메이션 | 수치·라벨 편집, 항목 추가/삭제 | 열 수, 애니메이션 속도·지연, 숫자 크기 |
| `card-list` | Card List | 클릭 가능한 카드 목록 | 카드 제목·설명·링크 편집 | — |
| `timeline` | Timeline | 타임라인/이력 표시 | 단계별 제목·날짜·내용 편집 | — |
| `swiper-slider` | Swiper Slider | 슬라이드 캐러셀 | 슬라이드별 콘텐츠 편집 | — |
| `media-grid` | Media Grid | 이미지/영상 그리드 | 미디어 소스 편집 | — |
| `media-slider` | Media Slider | 미디어 슬라이드쇼 | 미디어 소스 편집 | — |
| `animated-stats` | Animated Stats | 애니메이션 통계 | 수치·라벨 편집 | — |
| `hero-animation` | Hero Animation | 애니메이션 히어로 섹션 | 제목·부제목·버튼 편집 | — |
| `before-after-slider` | Before After Slider | 이미지 비교 슬라이더 | 전/후 이미지 편집 | — |
| `code` | Code Block | 구문 강조 코드 블록 | 코드 내용·언어 선택 | — |
| `video-embed` | Video Embed | YouTube/영상 임베드 | 영상 URL 편집 | — |
| `logo-loop` | Logo Loop | 로고 무한 스크롤 | 로고 이미지 편집 | — |
| `social-share` | Social Share | SNS 공유 버튼 | 공유 대상 편집 | — |

**수정 요청 예시:**
```
faq 컴포넌트에 항목 3개 추가하고, 첫 항목 기본으로 열려 있게 해줘
callout-box warning 스타일로 바꾸고 제목을 "주의사항"으로 바꿔줘
simple-stats 카운팅 애니메이션 속도를 2초 → 1초로 바꿔줘
card-list 카드에 뱃지(New) 표시 추가해줘
```

## 잘못된 요청 vs 잘된 요청

| ❌ 이렇게 하면 느림 | ✅ 이렇게 하면 빠름 |
|---|---|
| "뭔가 금융 관련 컴포넌트 하나 만들어줘" | "환전 신청 버튼 컴포넌트 만들어줘. 버튼 텍스트/색상 에디터에서 수정 가능하게" |
| "저거 좀 고쳐줘" | "quick-banking 아이콘 크기 48px → 36px으로 줄여줘" |
| "API 연동해줘" | "exchange-board에 /api/exchange로 환율 fetch 연동해줘. 응답: [{code, buy, sell, change}]" |
| "디자인 바꿔줘" | "loan-calculator 결과 영역 배경 회색 → 파란색 계열 그라데이션으로 바꿔줘" |
