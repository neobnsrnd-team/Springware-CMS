## 필드 성격 분류 요약

| 성격 | 설명 | 해당 필드 예시 |
|---|---|---|
| **단일 텍스트** | 컴포넌트당 1개 | 섹션 제목, 은행명, 버튼 레이블 |
| **반복** | 개수 가변 (추가/삭제 가능) | 메뉴 항목, 슬라이드, 지점 목록 |
| **고정** | 개수 고정 (변경 불가) | 계산기 탭, 필터 버튼, SNS 아이콘 |
| **JS 동적** | 런타임에 JS가 채워 넣음 | 환율 수치, 계산 결과, 슬라이드 카운터 |
